# 🛡️ FinSecure AI: Real-Time Financial Fraud Detection System

**FinSecure AI** is an enterprise-grade, end-to-end data pipeline designed to detect fraudulent financial transactions in real-time. Developed during my internship at **Amdocs Technologies** under the mentorship of **Chandan Mishra Sir**, this system integrates advanced machine learning models with robust data streaming architecture.

## 🚀 Key Features
* **Real-Time Data Streaming:** Leverages **Apache Kafka** and **Spark Streaming** to ingest and process transaction data with low latency.
* **Intelligent Fraud Detection:** Powered by high-performance ML algorithms, including **XGBoost** and **Random Forest**, for anomaly detection.
* **ETL Pipeline:** Automated data extraction, transformation, and loading (ETL) using **SQLite** and **PySpark**.
* **Interactive Dashboard:** A live **Streamlit** dashboard for real-time visualization of transaction trends, fraud heatmaps, and anomaly scores.
* **Automated Alerting:** Integrated SMTP-based alerting system to notify stakeholders instantly about high-risk transactions.

## 🛠️ Tech Stack
* **Programming:** Python, PySpark
* **Machine Learning:** Scikit-Learn, XGBoost, MLlib
* **Data Streaming:** Apache Kafka, Spark Streaming
* **Database:** SQLite
* **Visualization:** Streamlit
* **Deployment:** Hugging Face Spaces & Docker
